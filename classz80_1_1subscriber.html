Detailed Description

template<typename T>
class z80::subscriber< T >

Шаблон класса subscriber. Обеспечивает интерфейс Подписчика на событие.

Template Parameters
TТип, с которым работает класс.

Интерфейс Подписчика заключается в получении:

  • новых значений,
  • оповещения об отписке со стороны Издателя.
