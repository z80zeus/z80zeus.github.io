Detailed Description

Класс реализует функционал семафорной синхронизации потоков выполнения.

Работа класса построена на использовании мьютекса, условной переменных и счётчика post/wait-вызовов. Каждый вызов post увеличивает счётчик. Потоки, вызывающие post не блокируются никогда. Каждый вызов wait - уменьшает счётчик. Потоки, вызывающие wait блокируются только если значение счётчика семафора равно нулю уже при вызове функции wait.
