Detailed Description

Publisher строковых команд из строкового потока.

Класс является Издателем, для чего наследуется от шаблонного класса z80::publisher<T> с шаблонным параметром std::string: в данном проекте команды представляют собой строки.

void publisherCommandsFromStream::start ( )

Запуск чтения потока.

Эта функция блокирует вызывающий код, выполняет циклическое чтение потока и рассылку полученных команд. При закрытии потока - цикл завершается и выполняется отписка всех подписчиков. Фактически, данная функция является eventloop'ом программы.
