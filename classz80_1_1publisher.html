Detailed Description

template<typename T>
class z80::publisher< T >

Шаблонный класс publisher. Обеспечивает функционал подписки-отписки-оповещения.

Template Parameters
TТип, с которым работает класс.

Работает с подписчиками типа z80::subscriber<T>, рассылая им уведомления путём вызова метода z80::subscriber<T>::update().
